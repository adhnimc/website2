What if We Were to Adjust the Seat Counts in the 2024 Indonesian Legislative Election?

Politics
Author

Adhni Mulachela

Imagine we wanted to explore how different seat distributions could affect the representation of political parties across Daerah Pemilihan (Dapil), or electoral districts. By adjusting the number of seats in each district, we can simulate different scenarios to see how changes in seat counts might impact the political landscape â€” such as which parties end up with more or fewer representatives.

The Division Factors:

Division Factors and Their Operations
Division.Factor Explanation
0.125 The number of seats is multiplied by 8.
0.25 The number of seats is multiplied by 4.
0.5 The number of seats is multiplied by 2.
1 No change; this is the current seat count.
2 The number of seats is divided by 2.
4 The number of seats is divided by 4.
8 The number of seats is divided by 8.

Important Adjustment - The Ceiling Function:

Since seat counts must be whole numbers, any division that results in fractions is handled using the ceiling function. For example, if dividing a seat count like 3 by 2 results in 1.5, the ceiling function rounds the number up to 2. This ensures that the seat counts remain practical and feasible.

Why Use These Division Factors?

These factors help simulate how different seat allocations could impact the balance of political power across districts. By adjusting the seat count using these factors, we can test how having more or fewer seats affects the representation of political parties. For example, multiplying the seat count by 8 with the 0.125 factor, or dividing the seats by 2 with the 2 factor, allows us to observe how these changes would affect the number of representatives each party receives.
